# Code Book

This code book describes the different fields in `tidy.txt`.

## Subject and Activity

* `Subject` - The ID of the subject (Integer).
* `Activity` - The name of the activity performed by the subject (String):
      * `WALKING` 
      * `WALKING UPSTAIRS` 
      * `WALKING DOWNSTAIRS` 
      * `SITTING` 
      * `STANDING` 
      * `LAYING`
